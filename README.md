# 버비 로켓 체스게임
>증식하는 폰(pawn)을 조합해, 비숍(로켓), 나이트(유탄), 록(방패), 퀸(관통탄)을 만들어 왕을 지키고, 상대방의 왕을 잡아라!
>  ... 라는 간단한 게임입니다.

## 현재까지 완성된 목록
>1. 폰 증식 및 간단한 애니메이션
>2. 카메라 컨트롤(이동과 줌인/아웃)
>3. 타일 드래그로 주변 폰 조합

## 앞으로 할 내용
>1. 비숍 구현

>6. 아주 간단한 인공지능 추가
>7. 온라인 기능 추가 및 업로드
